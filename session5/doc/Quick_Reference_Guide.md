## ğŸ”§ Troubleshooting

### Document Processing Fails
**Problem:** Error during document upload
**Solutions:**
1. Check file format (PDF, DOCX, TXT only)
2. Ensure file is not corrupted
3. Try smaller file size (<50MB)
4. Check if PDF is text-based (not scanned image)

### Analysis Takes Too Long
**Problem:** Analysis stuck or very slow
**Solutions:**
1. Check internet connection
2. Wait for progress bar to complete (can take 2-5 minutes)
3. Refresh page and try again
4. Use smaller documents

### Chart Not Displaying
**Problem:** Blank space where chart should be
**Solutions:**
1. Scroll down to ensure chart is in view
2. Refresh page
3. Check browser console for errors
4. Try different browser

### Chat Not Responding
**Problem:** Chat assistant not generating responses
**Solutions:**
1. Ensure documents are processed first
2. Check question is clear and specific
3. Wait 10-15 seconds for response
4. Refresh page if stuck
5. Clear chat history and try again

### Export Button Not Working
**Problem:** Download doesn't start
**Solutions:**
1. Ensure analysis is complete
2. Check browser allows downloads
3. Try different export format
4. Refresh page and try again
